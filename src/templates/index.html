List of endpoints and their methods

  • Read to-dos:

    GET /todos

    Returns a list with all the to-do's in JSON.

    Try endpoint

  • Add to-do:

    POST /todos

    Adds a new to-do to the list.

    The body must have a JSON format, like this:

    
  {
    "label": "New to-do"
    "done": false,
  }
  • Delete specific to-do:

    DELETE /todos/:id

    Deletes a task by its index position, given in the endpoint as an integer.
